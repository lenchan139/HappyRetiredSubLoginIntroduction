# Facebook登入（iOS）
### 1.打開「樂活新中年」App後，撳左上角嘅「人仔icon（黑圈示）」
![step1](https://github.com/lenchan139/HappyRetiredSubLoginIntroduction/blob/master/iOS/Facebook/Screen%20Shot%202017-08-30%20at%206.02.49%20PM.png?raw=true)
### 2.選擇返「Facebook登入」
![step2](https://github.com/lenchan139/HappyRetiredSubLoginIntroduction/blob/master/iOS/Facebook/Screen%20Shot%202017-08-30%20at%206.04.00%20PM.png?raw=true)
### 3.若已登入，按下「繼續」即可
![step3](https://github.com/lenchan139/HappyRetiredSubLoginIntroduction/blob/master/iOS/Facebook/Screen%20Shot%202017-08-30%20at%206.06.18%20PM.png?raw=true)
### 3.1.若果未登入，請按照指示登入返你嘅Facebook
![step3.1](https://github.com/lenchan139/HappyRetiredSubLoginIntroduction/blob/master/iOS/Facebook/Screen%20Shot%202017-08-30%20at%206.06.02%20PM.png?raw=true)
### 4.若果登入成功，會返回主頁。
![step4](https://github.com/lenchan139/HappyRetiredSubLoginIntroduction/blob/master/iOS/Facebook/Screen%20Shot%202017-08-30%20at%206.10.25%20PM.png?raw=true)
### 5.若果喺第一次註冊，會要求 閣下註冊，只需跟住填寫簡單嘅個人資即可。
![step5](https://github.com/lenchan139/HappyRetiredSubLoginIntroduction/blob/master/iOS/Facebook/Screen%20Shot%202017-08-30%20at%206.06.29%20PM.png?raw=true)
